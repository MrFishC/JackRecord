# 三.数据结构基础
+ 数组和链表
  + 其它形式的数据结构都是基于这两者实现的（暂时不深入，略）
+ 数组和链表的优缺点（以下知识简单说明，面试的时候需要丰满一点，必须得会）
  + 数组
    + 顺序存储
    + 访问快速，增删慢
    + 增删的时间复杂度是O(N)
  + 链表
    + 链式存储
    + 增删快，访问慢
    + 增删的时间复杂度是O(1)  
   
+ 数组和链表的基本操作
  + 遍历
    + 线性：使用循环来处理
    + 非线性：使用递归
